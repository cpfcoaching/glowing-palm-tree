# Content Extraction

## Overview

This skill transforms existing long-form content (transcripts, videos, articles) into a systematic content pipeline for multiple platforms.

## When to Use This Skill

- Repurposing articles, podcast transcripts, or videos.
- Filling a content calendar from existing materials.
- Creating platform-specific Hooks/Angles from a core idea.

## Step-by-Step Process

### Phase 1: Topic Clarity

**Actions:**

- Identify platform targets (Newsletters, Substack Notes, X, LinkedIn, YouTube Shorts).
- Determine volume (how many pieces per platform).

### Phase 2: Insight Mining

**Actions:**

- Extract non-obvious insights and "AHA" moments.
- Select top 5-10 high-potential ideas.

### Phase 3: Platform Generation

**Actions:**

- Create platform tables including:
  - Title/Hook
  - Core Concept
  - Sub-points/Body
  - Call to Action (CTA)

## Anti-Patterns

❌ **Generic Summaries**: Simply summarizing the content instead of adapting it for specific platform mechanics.
